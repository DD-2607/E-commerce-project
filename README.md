# E-commerce-project
Fashion Store - E-Commerce Website

**Project Overview**

Fashion Store is a responsive e-commerce website built using HTML, CSS, and JavaScript. The website allows users to browse dress collections, search for products, and contact the store through a dedicated contact page.

**Features**

Home Page:
Attractive landing page showcasing featured dresses.
Responsive navigation bar.
New arrivals and product highlights.
Modern and user-friendly design.

Collections Page:
Displays a collection of dresses.
Users can browse available products.
Product cards with images and details.

Search Functionality:
Users can search for dresses by name.
Example: Searching for "White Dress" displays matching white dress products.
Implemented using JavaScript.

Contact Page:
Contact information including:
(Location
Phone Number
Email Address)
Contact section with Font Awesome icons.

**Responsive Design**
Mobile-friendly layout.
Hamburger menu for smaller screens.
Optimized for desktop, tablet, and mobile devices.

**Technologies Used**
HTML5
CSS3
JavaScript
Font Awesome

**How to Run the Project**
1.Download or clone the repository.
2.Open the project folder.
3.Open index.html in any web browser.
